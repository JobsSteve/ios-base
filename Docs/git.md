84. Git commands. 
==

## Git Commands

### Merge. Мерж веток. И

Мержим ветку upload в ветку в которой сейчас находимся

```git
git merge upload
```

### Checkout. Смена рабочей ветки.

Создаем новую ветку upload

```git
git checkout -b upload
```

Переключаемся на ветку upload и делаем ее рабочей

```git
git checkout upload
```
